

# 🚀 Projeto: API Node.js com MongoDB

## 📋 Sobre o Projeto

Este repositório contém o passo a passo da construção de uma **API RESTful** desenvolvida do zero. O objetivo principal é aplicar conceitos fundamentais de Back-End, manipulação de dados e integração com banco de dados em nuvem.

> **Nota:** Este projeto está atualmente **em desenvolvimento**. Novas funcionalidades e documentações são adicionadas regularmente.

-----

## 🛠️ Tecnologias Utilizadas

  * **Node.js**: Ambiente de execução para o JavaScript no servidor.
  * **Express**: Framework minimalista para gerenciar rotas e requisições HTTP.
  * **MongoDB Atlas**: Banco de dados NoSQL hospedado na nuvem.
  * **Mongoose**: Biblioteca de modelagem de dados para MongoDB e Node.js.
  * **Nodemon**: Ferramenta de auxílio ao desenvolvimento que reinicia o servidor automaticamente.

-----

  - [x] Configuração inicial do ambiente (npm init).
  - [x] Criação do servidor básico com Express.
